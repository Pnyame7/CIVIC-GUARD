CivicGuard

CivicGuard is an AI-powered mobile and web-based civic engagement platform designed to improve how citizens report and track community issues such as potholes,
flooding, broken streetlights, and waste management problems. The system connects citizens with local authorities for faster, smarter, and more transparent 
issue resolution.

Project Overview

CivicGuard allows citizens to:

Report civic issues using photos, descriptions, and GPS location.
Automatically categorize issues using AI.
Track complaint status in real time.
Receive updates from authorities.

Authorities can:

View and manage reports through a dashboard.
Visualize issues on interactive maps.
Detect duplicate reports using AI.
Analyze civic issue trends for decision-making.

 System Architecture

CivicGuard follows a 3-tier architecture:

Mobile App (Citizen Side)
Web Dashboard (Admin Side)
Backend System (AI + Database Layer)

Key Technologies:
Frontend: React Native (Expo), React.js
Backend: Node.js / Express
Database: SQLite (Drizzle ORM)
AI: Gemini API, NLP models
Maps: Google Maps API

 How It Works
Citizen opens the app
Logs in / registers
Captures photo of issue
Adds description
GPS automatically captures location
AI classifies the issue
Report is stored in database
Authorities receive and review report
Status updates are sent back to citizen
