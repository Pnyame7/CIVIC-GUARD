# Literature Review

The CivicGuard system is inspired by existing civic engagement platforms such as:

## FixMyStreet (UK)

Allows citizens to report local issues but lacks advanced AI integration.

## SeeClickFix (USA)

Provides community reporting and voting but limited automation.

## MyGOV (India)

Encourages citizen participation but lacks real-time emergency handling.

## Gap Identified

Existing systems lack:

- AI-based classification
- Duplicate detection
- Intelligent prioritization
- Integrated chatbot support

CivicGuard addresses these limitations using AI and modern web technologies.
